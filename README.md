# SaUI
SaUI，一个适用于 Desktop &amp; Mobile 的 CSS 框架

## Usage
#### NPM
```bash
npm install saui
```
#### Bower
```bash
bower install saui
```
#### HTML
```html
<link rel="stylesheet" type="text/css" href="[PATH]/css/sa.min.css">
<script src="[PATH]/js/jquery.sa.ui.min.js"></script>
```
